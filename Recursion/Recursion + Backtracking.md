`Recursion` is when a function **calls itself** to solve smaller instances of the same problem. It consists of a base case where recursion stops/returns.

`Backtracking` is a technique used to solve problems **recursively by trying all possibilities** and **undoing** choices (backtracking) that lead to a dead end.
